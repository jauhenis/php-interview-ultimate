# PHP Interview Questions & Answers

Welcome to the ultimate repository of PHP interview questions and answers.

## Structure

The questions are organized into 27 major categories, which you can find in the left navigation menu.

- **Categories:** 27 main topics (PHP Basics, OOP, Laravel, etc.)
- **Levels:** Each question is marked with its corresponding level (Junior, Middle, Senior, or Expert).
- **Detailed Answers:** In-depth guides for complex topics, accessible via the "Answers" menu link.

## How to use

Select a category from the sidebar to view all related questions. Each page contains questions from all experience levels for that specific topic.

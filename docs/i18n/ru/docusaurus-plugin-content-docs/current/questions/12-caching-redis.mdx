# 12. Кэширование и Redis

### Что такое кэширование?

<span className="badge badge--success margin-bottom--md">Junior</span>

**Ответ:** Сохранение данных во временном хранилище (например, в оперативной памяти) для более быстрого извлечения при последующих запросах.

---

### Redis и Memcached: сравнение

<span className="badge badge--primary margin-bottom--md">Middle</span>

**Ответ:**

- **Redis:** Поддерживает различные типы данных (строки, списки, множества и др.), персистентность и механизм pub/sub.
- **Memcached:** Простое хранилище «ключ-значение», в основном для простого кэширования.
  [Сравнение Redis и Memcached](../answers/caching.md)

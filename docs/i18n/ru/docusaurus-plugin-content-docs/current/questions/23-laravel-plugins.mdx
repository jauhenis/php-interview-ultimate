# 23. Плагины Laravel



### Какие наиболее популярные официальные плагины Laravel существуют?

<span className="badge badge--success margin-bottom--md">Junior</span>

**Ответ:** Laravel предоставляет обширную экосистему официальных пакетов для расширения функциональности. Среди наиболее широко используемых:
[Подробное руководство по плагинам Laravel](../answers/laravel_plugins.md)

- **Laravel Horizon:** Красивая панель управления и конфигурация через код для очередей на базе Redis.
- **Laravel Breeze:** Минималистичная реализация всех функций аутентификации Laravel.
- **Laravel Sanctum:** Легковесная система аутентификации для API и SPA.
- **Laravel Socialite:** Удобный интерфейс для OAuth-аутентификации через различных провайдеров (Google, GitHub и др.).
- **Laravel Sail:** Легковесный CLI для взаимодействия со стандартной Docker-конфигурацией Laravel.

---

### В чём назначение Laravel Horizon и Laravel Octane?

<span className="badge badge--primary margin-bottom--md">Middle</span>

**Ответ:**

- **Laravel Horizon:** Предоставляет панель для мониторинга пропускной способности задач, времени выполнения и ошибок в очередях Redis. Позволяет конфигурировать обработчики очередей через код.
- **Laravel Octane:** Значительно повышает производительность приложения, сохраняя его «инициализированным» в памяти с помощью высокопроизводительных серверов, таких как Swoole или RoadRunner.

---

### Что такое Laravel Livewire и Laravel Inertia?

<span className="badge badge--primary margin-bottom--md">Middle</span>

**Ответ:**

- **Laravel Livewire:** Full-stack фреймворк, позволяющий создавать динамические интерфейсы, используя только PHP и минимум JavaScript.
- **Laravel Inertia:** Позволяет создавать одностраничные приложения (SPA) с классической серверной маршрутизацией и контроллерами, используя современные фронтенд-фреймворки, такие как Vue или React.

---

### Сравните Laravel Pulse и Laravel Telescope.

<span className="badge badge--warning margin-bottom--md">Senior</span>

**Ответ:**

- **Laravel Pulse:** Предназначен для **мониторинга в реальном времени** состояния и производительности приложения в продакшене (например, медленные маршруты, высокая нагрузка на CPU).
- **Laravel Telescope:** **Помощник для отладки** в локальной разработке или на стейджинге, предоставляющий глубокий анализ запросов, исключений, логов, запросов к базе данных и многого другого.

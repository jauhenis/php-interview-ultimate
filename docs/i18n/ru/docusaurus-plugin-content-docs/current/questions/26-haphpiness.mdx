# 26. HaPHPiness - Лучшее в PHP

Этот раздел освещает современные возможности PHP и общую «радость» от экосистемы, вдохновленный ресурсом [haphpiness.com](https://haphpiness.com/).

### Что такое именованные аргументы (Named Arguments)?
<span className="badge badge--success margin-bottom--md">Junior</span>

**Ответ:** Именованные аргументы (появились в PHP 8.0) позволяют передавать значения по имени параметра, а не по позиции.
[См. Возможности PHP 8.0](../questions/07-php-7-8-new-features.mdx#что-такое-именованные-аргументы-named-arguments-появились-в-php-80)

---

### Что такое Union Types и Intersection Types?
<span className="badge badge--success margin-bottom--md">Junior</span>

**Ответ:** Union Types (8.0) позволяют указывать несколько типов для параметра; Intersection Types (8.1) требуют соответствия значения сразу нескольким ограничениям типов.
[См. Возможности PHP 8.0](../questions/07-php-7-8-new-features.mdx#что-такое-union-types-появились-в-php-80)

---

### Что такое перечисления (Enums)?
<span className="badge badge--success margin-bottom--md">Junior</span>

**Ответ:** Перечисления (8.1) обеспечивают типобезопасность для фиксированного набора именованных констант.
[См. Возможности PHP 8.1](../questions/07-php-7-8-new-features.mdx#что-такое-перечисления-enums-появились-в-php-81)

---

### Что такое str_contains(), str_starts_with() и str_ends_with()?
<span className="badge badge--success margin-bottom--md">Junior</span>

**Ответ:** Это консистентные и читаемые функции для стандартных операций со строками, добавленные в PHP 8.0.
[Подробный гайд HaPHPiness](../answers/haphpiness.md#string-functions)

---

### Что такое распаковка массивов со строковыми ключами?
<span className="badge badge--success margin-bottom--md">Junior</span>

**Ответ:** Начиная с PHP 8.1, оператор расширения (`...`) поддерживает строковые ключи при слиянии массивов.
[Подробный гайд HaPHPiness](../answers/haphpiness.md#array-unpacking)

---

### Что такое встроенный сервер разработки?
<span className="badge badge--success margin-bottom--md">Junior</span>

**Ответ:** PHP включает простой веб-сервер для целей разработки, запускаемый командой `php -S localhost:8000`.
[Подробный гайд HaPHPiness](../answers/haphpiness.md#dev-server)

---

### Что такое стрелочные функции (Arrow Functions)?
<span className="badge badge--success margin-bottom--md">Junior</span>

**Ответ:** Введенные в PHP 7.4, они предлагают краткий синтаксис для коротких замыканий с автоматическим захватом переменных.
[Подробный гайд HaPHPiness](../answers/haphpiness.md#arrow-functions)

---

### Что такое выражения Match?
<span className="badge badge--success margin-bottom--md">Junior</span>

**Ответ:** Более безопасная и строгая альтернатива `switch`, которая возвращает значение (PHP 8.0).
[Подробный гайд HaPHPiness](../answers/haphpiness.md#match-expressions)

---

### Что такое оператор объединения с null (Null Coalescing Operator)?
<span className="badge badge--success margin-bottom--md">Junior</span>

**Ответ:** Оператор `??` (и присваивание `??=`) обеспечивает чистый способ обработки null-значений и дефолтов.
[Подробный гайд HaPHPiness](../answers/haphpiness.md#null-coalescing)

---

### Что такое разделители в числовых литералах?
<span className="badge badge--success margin-bottom--md">Junior</span>

**Ответ:** Использование нижнего подчеркивания для улучшения читаемости больших чисел (например, `1_000_000`).
[Подробный гайд HaPHPiness](../answers/haphpiness.md#numeric-separators)

---

### Что такое файберы (Fibers)?
<span className="badge badge--primary margin-bottom--md">Middle</span>

**Ответ:** Легковесные корутины для неблокирующего ввода-вывода (PHP 8.1).
[См. Возможности PHP 8.1](../questions/07-php-7-8-new-features.mdx#что-такое-fibers-появились-в-php-81)

---

### Что такое атрибуты (Attributes)?
<span className="badge badge--primary margin-bottom--md">Middle</span>

**Ответ:** Нативная поддержка метаданных (аннотаций) в движке PHP начиная с версии 8.0.
[Подробный гайд HaPHPiness](../answers/haphpiness.md#attributes)

---

### Что такое первоклассный синтаксис callable (First-class Callable Syntax)?
<span className="badge badge--primary margin-bottom--md">Middle</span>

**Ответ:** Чистый способ создания `Closure` из функции или метода с использованием `(...)`.
[См. Возможности PHP 8.1](../questions/07-php-7-8-new-features.mdx#что-такое-синтаксис-callable-первого-класса-появился-в-php-81)

---

### Что такое оператор Nullsafe?
<span className="badge badge--primary margin-bottom--md">Middle</span>

**Ответ:** Оператор `?->` позволяет безопасно обращаться к методам или свойствам потенциально null-объектов.
[См. Возможности PHP 8.0](../questions/07-php-7-8-new-features.mdx#что-такое-оператор-nullsafe-появился-в-php-80)

---

### Что такое хуки свойств (Property Hooks)?
<span className="badge badge--primary margin-bottom--md">Middle</span>

**Ответ:** Логика для `get` и `set`, определяемая прямо при объявлении свойства (PHP 8.4).
[См. Возможности PHP 8.4](../questions/07-php-7-8-new-features.mdx#что-такое-хуки-свойств-появились-в-php-84)

---

### Что такое асимметричная видимость (Asymmetric Visibility)?
<span className="badge badge--primary margin-bottom--md">Middle</span>

**Ответ:** Разная область видимости для чтения и записи свойства (например, `public private(set)`).
[См. Возможности PHP 8.4](../questions/07-php-7-8-new-features.mdx#что-такое-асимметричная-видимость-появилась-в-php-84)

---

### Что такое конвейерный оператор (Pipe Operator)?
<span className="badge badge--primary margin-bottom--md">Middle</span>

**Ответ:** Способ выстраивания цепочек вызовов функций слева направо с помощью `|>` (PHP 8.5).
[См. Возможности PHP 8.5](../questions/07-php-7-8-new-features.mdx#что-такое-оператор-pipe-появился-в-php-85)

---

### Что такое array_is_list()?
<span className="badge badge--primary margin-bottom--md">Middle</span>

**Ответ:** Функция для проверки, является ли массив списком с последовательными ключами, начиная с 0.
[Подробный гайд HaPHPiness](../answers/haphpiness.md#array-is-list)

---

### Что такое WeakMaps?
<span className="badge badge--primary margin-bottom--md">Middle</span>

**Ответ:** Карты, где ключами являются объекты, которые не препятствуют удалению этих объектов сборщиком мусора.
[Подробный гайд HaPHPiness](../answers/haphpiness.md#weakmap)

---

### Что такое именованные группы захвата в preg_match?
<span className="badge badge--primary margin-bottom--md">Middle</span>

**Ответ:** Использование `(?P<name>...)` в регулярных выражениях для получения именованных совпадений.
[Подробный гайд HaPHPiness](../answers/haphpiness.md#named-capture-groups)

---

### Что такое оператор «космический корабль» (Spaceship Operator)?
<span className="badge badge--primary margin-bottom--md">Middle</span>

**Ответ:** Оператор `<=>` для трехстороннего сравнения, возвращающий -1, 0 или 1.
[Подробный гайд HaPHPiness](../answers/haphpiness.md#spaceship-operator)

---

### Что такое распаковка массива с ключами?
<span className="badge badge--primary margin-bottom--md">Middle</span>

**Ответ:** Извлечение конкретных значений из ассоциативного массива с использованием синтаксиса `[...]`.
[Подробный гайд HaPHPiness](../answers/haphpiness.md#array-destructuring)

---

### Что такое #[\NoDiscard]?
<span className="badge badge--warning margin-bottom--md">Senior</span>

**Ответ:** Атрибут, который выдает предупреждение, если возвращаемое значение функции игнорируется (PHP 8.5).
[Подробный гайд HaPHPiness](../answers/haphpiness.md#nodiscard)

---

### Что такое Fatal Error Backtraces (трассировка стека при фатальных ошибках)?
<span className="badge badge--warning margin-bottom--md">Senior</span>

**Ответ:** Начиная с PHP 8.5, фатальные ошибки (например, таймауты) включают полную трассировку стека (backtrace) для упрощения отладки.
[Подробный гайд HaPHPiness](../answers/haphpiness.md#fatal-error-backtraces)

---

### Что такое расширение URI?
<span className="badge badge--warning margin-bottom--md">Senior</span>

**Ответ:** Встроенное расширение для парсинга URL, работающее с иммутабельными объектами и соответствующее стандартам (PHP 8.5).
[Подробный гайд HaPHPiness](../answers/haphpiness.md#uri-extension)

---

### Что такое FFI (Foreign Function Interface)?
<span className="badge badge--warning margin-bottom--md">Senior</span>

**Ответ:** Позволяет вызывать функции C и использовать структуры данных C напрямую из PHP (7.4+).
[Подробный гайд HaPHPiness](../answers/haphpiness.md#ffi)

---

### Что такое PHPStan и Psalm?
<span className="badge badge--warning margin-bottom--md">Senior</span>

**Ответ:** Инструменты статического анализа, обеспечивающие уровень типобезопасности как в TypeScript для PHP-проектов.
[Подробный гайд HaPHPiness](../answers/haphpiness.md#static-analysis)

---

<span className="badge badge--warning margin-bottom--md">Senior</span>

**Ответ:** Фреймворк для создания нативных десктопных и мобильных приложений на базе PHP и Laravel.

# 29. Регулярные выражения (Regexp)



### Что такое Regex?
<span className="badge badge--success margin-bottom--md">Junior</span>

**Ответ:** Regex (Регулярное выражение) — это мощный инструмент для сопоставления текста, поиска и манипуляций. В PHP он реализован через библиотеку PCRE.
[Подробное руководство по Regex](../answers/regexp.md#7-продвинутые-возможности-pcre)

---

### Почему Regex — это хорошо?
<span className="badge badge--success margin-bottom--md">Junior</span>

**Ответ:** Это краткий, мощный и чрезвычайно эффективный инструмент при правильном написании. Он позволяет описывать сложные текстовые шаблоны одной строкой.
[Подробное руководство по Regex](../answers/regexp.md#10-элементы-хорошего-стиля-regex)

---

### Как использовать Regex в PHP?
<span className="badge badge--success margin-bottom--md">Junior</span>

**Ответ:** PHP использует функции `preg_*`, такие как `preg_match()`, `preg_match_all()` и `preg_replace()`. Шаблоны должны быть заключены в разделители (например, `/шаблон/`).
[Подробное руководство по Regex](../answers/regexp.md#11-функции-regex-в-php-подробно)

---

### В чем разница между `preg_match` и `preg_match_all`?
<span className="badge badge--success margin-bottom--md">Junior</span>

**Ответ:** `preg_match` останавливается после первого совпадения, в то время как `preg_match_all` находит все вхождения в строке.
[Подробное руководство по Regex](../answers/regexp.md#11-функции-regex-в-php-подробно)

---

### Как создать сложный Regex пошагово?
<span className="badge badge--primary margin-bottom--md">Middle</span>

**Ответ:** 1. Определите, что игнорировать (кавычки, теги). 2. Определите целевой шаблон. 3. Объедините с помощью «Лучшего трюка» (`Игнор|Цель|(Захват)`). 4. Оптимизируйте по AGRA (Якорь, Жадность, Повтор, Атомарность).
[Подробное руководство по Regex](../answers/regexp.md#9-пошаговое-руководство-создание-сложного-regex)

---

### Что такое «Лучший трюк Regex в истории»?
<span className="badge badge--primary margin-bottom--md">Middle</span>

**Ответ:** Это техника сопоставления паттернов с исключением конкретных контекстов (например, кавычек или комментариев) с помощью простого шаблона `Игнор|Цель|(Захват)`.
[Подробное руководство по Regex](../answers/regexp.md#8-лучший-трюк-regex-в-истории-исключение-контекста)

---

### Что такое `preg_filter`?
<span className="badge badge--primary margin-bottom--md">Middle</span>

**Ответ:** Это аналог `preg_replace`, но он возвращает только те элементы, которые действительно соответствуют шаблону, фактически фильтруя входной массив.
[Подробное руководство по Regex](../answers/regexp.md#11-функции-regex-в-php-подробно)

---

### Что такое атомарные группы и зачем они нужны?
<span className="badge badge--warning margin-bottom--md">Senior</span>

**Ответ:** Группы `(?>...)` предотвращают возврат (backtracking). Они «фиксируют» совпадение, не позволяя движку пробовать альтернативные пути, что предотвращает «катастрофический возврат» и повышает производительность.
[Подробное руководство по Regex](../answers/regexp.md#атомарные-группы)

---

### Хорош ли ChatGPT в регулярных выражениях?
<span className="badge badge--warning margin-bottom--md">Senior</span>

**Ответ:** Хотя ChatGPT полезен для черновиков, ему часто не хватает глубоких знаний. Он может создавать неэффективные шаблоны («dot-star soup») или допускать синтаксические ошибки. Всегда требуется проверка экспертом.
[Подробное руководство по Regex](../answers/regexp.md#12-хорош-ли-chatgpt-в-регулярных-выражениях)

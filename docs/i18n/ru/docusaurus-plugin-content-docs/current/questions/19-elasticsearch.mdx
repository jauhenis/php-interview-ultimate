# 19. Elasticsearch

### Что такое Elasticsearch и каковы его основные возможности?

<span className="badge badge--primary margin-bottom--md">Middle</span>

**Ответ:** Распределённый RESTful-движок для поиска и аналитики, построенный на основе Apache Lucene. Обеспечивает быстрый поиск по большим наборам данных.
[Возможности Elasticsearch](../answers/elasticsearch.md)

---

### Что такое инвертированный индекс (Inverted Index)?

<span className="badge badge--primary margin-bottom--md">Middle</span>

**Ответ:** Основная структура данных Elasticsearch, которая сопоставляет термины (слова) с документами, в которых они встречаются, что обеспечивает очень быстрый поиск.

---

### Что такое анализаторы (Analyzers)?

<span className="badge badge--primary margin-bottom--md">Middle</span>

**Ответ:** Компоненты, обрабатывающие текст при индексации или поиске. Они состоят из символьного фильтра (Character Filter), токенизатора (Tokenizer) и фильтра токенов (Token Filter).

# Связи между таблицами PostgreSQL

PostgreSQL поддерживает стандартные соединения SQL и операции над множествами.

## 1. Соединения (Joins)

### INNER JOIN
Возвращает строки, если есть совпадение в обеих таблицах.

### LEFT [OUTER] JOIN
Возвращает все строки из левой таблицы и совпадающие строки из правой таблицы.

### RIGHT [OUTER] JOIN
Возвращает все строки из правой таблицы и совпадающие строки из левой таблицы.

### FULL [OUTER] JOIN
Возвращает все строки, если есть совпадение в левой или правой таблице.

### CROSS JOIN
Возвращает декартово произведение двух таблиц.

## 2. Операции над множествами

### UNION
Объединяет результат двух или более операторов SELECT (удаляет дубликаты). Используйте `UNION ALL`, чтобы сохранить дубликаты.

### INTERSECT
Возвращает только те строки, которые присутствуют в обоих результирующих наборах.

### EXCEPT
Возвращает строки из первого результирующего набора, которых нет во втором.

### 1. Two Sum
<span className="badge badge--success margin-bottom--md">Junior</span>

**Описание:** Дан массив целых чисел nums и целое число target, верните индексы двух чисел таких, чтобы их сумма была равна target.

**Код:**
```php
class Solution {
    /**
     * @param Integer[] $nums
     * @param Integer $target
     * @return Integer[]
     */
    function twoSum($nums, $target) {
        $d = [];
        foreach ($nums as $i => $x) {
            $y = $target - $x;
            if (isset($d[$y])) {
                return [$d[$y], $i];
            }
            $d[$x] = $i;
        }
    }
}
```

### 9. Palindrome Number
<span className="badge badge--success margin-bottom--md">Junior</span>

**Описание:** Дано целое число x, верните true, если x является палиндромом, и false в противном случае.

**Код:**
```php
class Solution {
    /**
     * @param Integer $x
     * @return Boolean
     */
    function isPalindrome($x) {
        if ($x < 0 || ($x && $x % 10 == 0)) {
            return false;
        }
        $y = 0;
        while ($x > $y) {
            $y = $y * 10 + ($x % 10);
            $x = (int) ($x / 10);
        }
        return $x == $y || $x == (int) ($y / 10);
    }
}
```

### 13. Roman to Integer
<span className="badge badge--success margin-bottom--md">Junior</span>

**Описание:** Римские цифры представлены семью различными символами: I, V, X, L, C, D и M. Дано римское число, преобразуйте его в целое число.

**Код:**
```php
class Solution {
    /**
     * @param String $s
     * @return Integer
     */
    function romanToInt($s) {
        $d = [
            'I' => 1,
            'V' => 5,
            'X' => 10,
            'L' => 50,
            'C' => 100,
            'D' => 500,
            'M' => 1000,
        ];
        $ans = 0;
        $len = strlen($s);

        for ($i = 0; $i < $len - 1; $i++) {
            if ($d[$s[$i]] < $d[$s[$i + 1]]) {
                $ans -= $d[$s[$i]];
            } else {
                $ans += $d[$s[$i]];
            }
        }

        $ans += $d[$s[$len - 1]];
        return $ans;
    }
}
```

### 14. Longest Common Prefix
<span className="badge badge--success margin-bottom--md">Junior</span>

**Описание:** Напишите функцию для поиска самой длинной строки общего префикса среди массива строк.

**Код:**
```php
class Solution {
    /**
     * @param String[] $strs
     * @return String
     */
    function longestCommonPrefix($strs) {
        $rs = '';
        for ($i = 0; $i < strlen($strs[0]); $i++) {
            for ($j = 1; $j < count($strs); $j++) {
                if ($strs[0][$i] != $strs[$j][$i]) {
                    return $rs;
                }
            }
            $rs = $rs . $strs[0][$i];
        }
        return $rs;
    }
}
```

### 20. Valid Parentheses
<span className="badge badge--success margin-bottom--md">Junior</span>

**Описание:** Дана строка s, содержащая только символы '(', ')', '{', '}', '[' и ']', определите, является ли входная строка допустимой.

**Код:**
```php
class Solution {
    /**
     * @param string $s
     * @return boolean
     */

    function isValid($s) {
        $stack = [];
        $brackets = [
            ')' => '(',
            '}' => '{',
            ']' => '[',
        ];

        for ($i = 0; $i < strlen($s); $i++) {
            $char = $s[$i];
            if (array_key_exists($char, $brackets)) {
                if (empty($stack) || $stack[count($stack) - 1] !== $brackets[$char]) {
                    return false;
                }
                array_pop($stack);
            } else {
                array_push($stack, $char);
            }
        }
        return empty($stack);
    }
}
```

### 21. Merge Two Sorted Lists
<span className="badge badge--success margin-bottom--md">Junior</span>

**Описание:** Слейте два отсортированных связанных списка и верните результат в виде отсортированного списка.

**Код:**
```php
/**
 * Definition for a singly-linked list.
 * class ListNode {
 *     public $val = 0;
 *     public $next = null;
 *     function __construct($val = 0, $next = null) {
 *         $this->val = $val;
 *         $this->next = $next;
 *     }
 * }
 */
class Solution {
    /**
     * @param ListNode $list1
     * @param ListNode $list2
     * @return ListNode
     */
    function mergeTwoLists($list1, $list2) {
        if (is_null($list1)) {
            return $list2;
        }
        if (is_null($list2)) {
            return $list1;
        }
        if ($list1->val <= $list2->val) {
            $list1->next = $this->mergeTwoLists($list1->next, $list2);
            return $list1;
        } else {
            $list2->next = $this->mergeTwoLists($list1, $list2->next);
            return $list2;
        }
    }
}
```

### 26. Remove Duplicates from Sorted Array
<span className="badge badge--success margin-bottom--md">Junior</span>

**Описание:** Дан массив целых чисел nums, отсортированный в неубывающем порядке, удалите дубликаты на месте так, чтобы каждый уникальный элемент появлялся только один раз.

**Код:**
```php
class Solution {
    /**
     * @param Integer[] $nums
     * @return Integer
     */
    function removeDuplicates(&$nums) {
        $k = 0;
        foreach ($nums as $x) {
            if ($k == 0 || $x != $nums[$k - 1]) {
                $nums[$k++] = $x;
            }
        }
        return $k;
    }
}
```

### 35. Search Insert Position
<span className="badge badge--success margin-bottom--md">Junior</span>

**Описание:** Дан отсортированный массив различных целых чисел и целевое значение, верните индекс, если цель найдена. Если нет, верните индекс, в котором она была бы, если бы была вставлена по порядку.

**Код:**
```php
class Solution {
    /**
     * @param Integer[] $nums
     * @param Integer $target
     * @return Integer
     */
    function searchInsert($nums, $target) {
        $l = 0;
        $r = count($nums);
        while ($l < $r) {
            $mid = $l + $r >> 1;
            if ($nums[$mid] >= $target) {
                $r = $mid;
            } else {
                $l = $mid + 1;
            }
        }
        return $l;
    }
}
```

### 100. Same Tree

**Описание:** Даны корни двух бинарных деревьев p и q, напишите функцию, чтобы проверить, являются ли они одинаковыми или нет.

**Код:**
```php
/**
 * Definition for a binary tree node.
 * class TreeNode {
 *     public $val = null;
 *     public $left = null;
 *     public $right = null;
 *     function __construct($val = 0, $left = null, $right = null) {
 *         $this->val = $val;
 *         $this->left = $left;
 *         $this->right = $right;
 *     }
 * }
 */
class Solution {
    /**
     * @param TreeNode $p
     * @param TreeNode $q
     * @return Boolean
     */
    function isSameTree($p, $q) {
        if ($p == null && $q == null) {
            return true;

        if ($p == null || $q == null) {
            return false;

        if ($p->val != $q->val) {
            return false;
        }
        return $this->isSameTree($p->left, $q->left) && $this->isSameTree($p->right, $q->right);
```

### 125. Valid Palindrome
<span className="badge badge--success margin-bottom--md">Junior</span>

**Описание:** Фраза является палиндромом, если после преобразования всех заглавных букв в строчные и удаления всех небуквенно-цифровых символов она читается одинаково вперед и назад.

**Код:**
```php
class Solution {
    /**
     * @param String $s
     * @return Boolean
     */
    function isPalindrome($s) {
        $regex = '/[a-z0-9]/';
        $s = strtolower($s);
        preg_match_all($regex, $s, $matches);
        if ($matches[0] == null) {
            return true;
        }
        $len = floor(count($matches[0]) / 2);
        for ($i = 0; $i < $len; $i++) {
            if ($matches[0][$i] != $matches[0][count($matches[0]) - 1 - $i]) {
                return false;
            }
        }
        return true;
    }
}
```

```php
    function __construct($nums) {
```

```php
        foreach ($nums1 as $x) {
            if (isset($cnt[$x])) {
                $cnt[$x]++;

                $cnt[$x] = 1;

            if (isset($cnt[$x]) && $cnt[$x] > 0) {

            }

    }
}
```

```php
        $cnt = 0;
        for ($i = 0; $i < count($arr); $i++) {
            if (strlen($arr[$i]) != 0) {
                $cnt++;
            }
        }
        return $cnt;
    }
}
```

### 485. Max Consecutive Ones
<span className="badge badge--success margin-bottom--md">Junior</span>

**Описание:** Дан бинарный массив nums, верните максимальное количество последовательных единиц в массиве.

**Код:**
```php
class Solution {
    /**
     * @param Integer[] $nums
     * @return Integer
     */
    function findMaxConsecutiveOnes($nums) {
        $ans = $cnt = 0;

        foreach ($nums as $x) {
            if ($x == 1) {
                $cnt += 1;
                $ans = max($ans, $cnt);
            } else {
                $cnt = 0;
            }
        }

        return $ans;
    }
}
```

### 509. Fibonacci Number
<span className="badge badge--success margin-bottom--md">Junior</span>

**Описание:** Числа Фибоначчи, обычно обозначаемые F(n), образуют последовательность, в которой каждое число является суммой двух предыдущих, начиная с 0 и 1.

**Код:**
```php
class Solution {
    /**
     * @param Integer $n
     * @return Integer
     */
    function fib($n) {
        $a = 0;
        $b = 1;
        for ($i = 0; $i < $n; $i++) {
            $temp = $a;
            $a = $b;
            $b = $temp + $b;
        }
        return $a;
    }
}
```
